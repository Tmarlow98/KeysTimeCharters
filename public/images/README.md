## Specs
- Export `.webp` where you can; keep a `.jpg` fallback at the same name.
- 1600px wide is plenty for cards, 2400px wide for the hero.
- Aim for < 250 KB per gallery image, < 350 KB for the hero.
- Compress with [Squoosh](https://squoosh.app) or `sharp` before committing.
